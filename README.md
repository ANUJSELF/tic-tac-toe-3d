# Tic-Tac-Toe 3D 🎮

A modern, interactive **3D Tic-Tac-Toe** game built with JavaScript, React, and Three.js.

## 🚀 Live Demo
🔗 **Play the game now:** [Tic-Tac-Toe 3D](https://indianmuseums.netlify.app/)

```sh
